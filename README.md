# Django College Management System

This is a Django-based web application that manages college departments.

## Features

- Add Department
- View Department List
- Edit Department
- Delete Department
- Admin Panel Support

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite
- Git & GitHub

## How to Run

1. Clone the repository
2. Run: python manage.py migrate
3. Run: python manage.py runserver
4. Open http://127.0.0.1:8000/
